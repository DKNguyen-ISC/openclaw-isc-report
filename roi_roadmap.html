Phased Execution Roadmap

0
Pre-PoC Preparation
Jan–Mar 2026

Validate OpenClaw local setup, configure ISC Sheets/BQ access permissions, define PoC success KPIs with Kent.

  • Install OpenClaw on isolated VM (no prod access)
  • Define Ngàn Chì BOM cluster as PoC scope (357 BOMs)
  • Establish baseline: manual PIC session timing + savings rate
  • KPI sign-off: accuracy ≥95%, savings ≥1%, runtime ≤PIC baseline
1
PoC: Ngàn Chì Pilot
Q2 2026 · 8 Weeks

First live run with human oversight. OpenClaw executes P1–P3 analysis and Gate B–E validation on read-only shadow mode.

  • Shadow execution alongside PIC (no auto-write)
  • Compare OpenClaw vs PIC decisions — accuracy matrix
  • Identify hallucination patterns, retrain prompt chains
  • Go/No-Go gate: ≥90% accuracy on Gate B–E
2
Phase 3 Augmentation: M3 PUBLIC Agg
Q3 2026 · 12 Weeks

OpenClaw takes over PUBLIC aggregation sessions autonomously. PIC supervises exception escalations only (Gate F).

  • Auto-write enabled for Gate B–D approvals (≥95% confidence)
  • Flight Recorder audit log integration with ISC Logger
  • Weekly savings report auto-generated to CEO dashboard
  • Target: 5x session throughput vs Q1 baseline
3
Phase 4 Full Automation: PRIVATE Agg + M4
Q4 2026–Q2 2027

Extend to PRIVATE aggregation and M4 issuance preview. Internal Llama3 model replaces external API for cost optimization.

  • PRIVATE agg automation with MOQ enforcement
  • M4 issuance preview trigger from OpenClaw output
  • Llama3 fine-tuned on ISC Phase2/3 data deployed
  • 10x session volume achieved · $500k+ annual savings verified
