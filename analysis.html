The Logic — ISC Shortage Engine

BigQuery SQL · shortage_engine.sql 
-- ISC Shortage Engine (Google BigQuery)
SELECT
  Part_ID,
  Part_Name,
  Demand_This_Week      AS Demand,
  Current_Stock         AS Stock,
  On_The_Way_PO         AS PO_Supply,
  (Demand - Stock - On_The_Way_PO) AS SHORTAGE
FROM SCM_Database
WHERE (Demand - Stock - On_The_Way_PO) > 0
ORDER BY SHORTAGE DESC;
